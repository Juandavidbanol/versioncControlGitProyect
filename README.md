# Proyecto Control de Versiones

Este proyecto es un ejemplo de implementación de un sistema de control de versiones usando Git y GitHub.  
Se incluyen varias historias de usuario para demostrar funcionalidades básicas.

## Historias de usuario implementadas
1. Mostrar mensaje de bienvenida
2. Archivo de configuración básica
3. Mostrar fecha actual
4. Botón que cambia de color al hacer clic
5. Registrar log simple en consola
6. Crear README básico
7. Validación mínima en un formulario

## Cómo ejecutar
1. Abrir el archivo `index.html` en un navegador.
2. Interactuar con los elementos de la página (botón, mensajes, etc.).
3. Ver la consola para los logs de eventos.
